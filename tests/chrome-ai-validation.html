🧪 Chrome AI Prompt API - 价格提取验证测试

测试目的

验证Chrome AI Prompt API能否从各大电商网站准确提取商品价格、标题和其他关键信息。

注意: 此测试需要在支持Chrome AI的浏览器中运行（Chrome 128+ with AI flags enabled）
